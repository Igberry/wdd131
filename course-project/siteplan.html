EduLearn Site Plan

Site Name

EduLearn - This name represents an online learning platform focused on providing technical and digital education courses such as web development, app development, and cybersecurity. The name "EduLearn" is chosen to signify both 'education' and 'learning' in a digital format.

Site Purpose

The purpose of the EduLearn website is to provide an easy-to-navigate platform where users can find various technology courses. The website offers access to online courses, information about the platform, instructors, and a login section for registered students.

Scenarios

  • Scenario 1: How can I sign up for the web development course?
  • Scenario 2: Where can I find the curriculum for mobile app development?

Color Schema

The color scheme for EduLearn consists of two primary colors:

  • #3498db - Used for the header, footer, navigation bar, and call-to-action buttons.
  • #f4f4f4 - Used for the background of the main content sections to provide a clean, minimal look.

Typography

EduLearn uses two main fonts:

  • Arial - This is the main font for body text, chosen for its readability and modern look.
  • Georgia - This font is used for headings (h1, h2) to give a more distinguished look to titles and section headers.
